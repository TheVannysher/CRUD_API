







// Two space-separated integers: number of digits d and sum of digits s.
// Output
// Smallest and largest numbers separated by space.
// Numbers are guaranteed to be < 2^63 and should not have leading 0s.
// There is always existing solution.
// Constraints
// 1 ≤ d ≤ 18
// 1 ≤ s ≤ 150
// Example
// Input

// 4 5

// Output

// 1004 5000


// 5 20

// Output

// 10199 99110
